## What it is

SYNTHIA is not a portfolio. It is an entire operating system.

Visitors boot into a fictional AI called **SYNTHIA**, enter a neon-soaked megacity, and physically navigate through 7 districts — each one revealing a different chapter of Rumman Khan's career:

| District | What lives here |
|---|---|
| **01 · Identity Core** | Central AI orb with operator profile |
| **02 · Education Nexus** | Academic timeline & future vision |
| **03 · Corporate Sector** | Kotak Mahindra · FireAI · Bluestock · Reliance Smart Money towers with KPI dashboards |
| **04 · The Data Center** | Skills as holograms + live SQL query stream |
| **05 · Research Lab** | Emotion Detection · Wildfire Prediction · Vaultwise · RegradED |
| **06 · Achievement Tower** | Goldman Sachs · IIM Bangalore · IIT Bombay · leadership & competitions |
| **07 · Mission Control** | Recruiter uplink terminal — LinkedIn, GitHub, resume, email, direct line |

## Features

- 🎬 **Cinematic boot sequence** — terminal typewriter, glitch title, camera zoom-in
- 🌆 **Animated cyberpunk city** — SVG skyline, rain, particles, flying vehicles, volumetric fog
- 🚀 **Missile-strike navigation** — press `I · E · C · D · R · A · M` to launch a missile at that district; it explodes on impact and opens the panel
- 🎵 **Procedural synthwave audio** — Web Audio API generated ambient pad + UI SFX (no downloaded assets)
- 📡 **Live HUD** — radar mini-map, mission log, clock, sound toggle, HYPER-NET status
- 📱 **Fully responsive** — desktop gets the cinematic city, mobile gets a clean tap-to-open card stack
- ♿ **Keyboard-first navigation** — every district reachable by a single key

## Tech stack

- **Framework:** Next.js 15 (App Router)
- **UI:** React + Tailwind CSS + shadcn/ui + Lucide icons
- **Animation:** Framer Motion + custom CSS keyframes
- **Audio:** Web Audio API (procedural)
- **Fonts:** Orbitron · Rajdhani · Share Tech Mono
- **Deployment:** Vercel

## Run locally
